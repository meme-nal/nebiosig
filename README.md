# nebiosig

**nebiosig** is a C++ library for processing biomedical signals

## Installation

```shell
git clone https://github.com/meme-nal/nebiosig.git
cd nebiosig
```

```shell
mkdir build
cd build
```

```shell
cmake -S ..
cmake --build .
sudo make install
```

## Usage

To use this library you have to include:

```cpp
#include <nebiosig/nebiosig.h>
```
